## Rooms and Interactions

### Start Room (Lobby)

- **Character**: Cleaner

## Jody

- **Items**:

### End Room (Basement)
## Jody
- **Puzzle**: Use items from previous rooms to navigate in the dark and rescue the kidnapped girl.
- **Items**: 
  - 
- **Character**: Kidnapped girl

- **Interactions**

  - Use fire extinguisher + the power outage to knock out kidnapper.

## Conclusion
The game's design focuses on engaging puzzles that are logically connected, with each room's elements contributing to the overarching narrative. The UI will present clear interaction options, combining narrative text with point-and-click mechanics for a smooth gameplay experience.
