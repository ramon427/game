1. De knoppen zijn duidelijk, in hotelroom4 kan er een aanpassing gemaakt worden: examine en look around valt onder 1 button. Dus de teksten kunnen bij examine worden samengevoegd.

2. Ja, je kan vanuit de lobby naar hotelroom4 gaan, dit doe je naar aanleiding van een gesprek met de employee.

3. Ja, je kan zeker praten met de employee, je krijgt de keuze om uiteindelijk te kiezen tussen hotelkaart 3 of 4. Gebasseerd op je keuze krijg je een andere verhaallijn wat heel goed is.

4. Ik begrijp makkelijk wat ik moet doen doormiddel van de afbeeldingen en teksten die eronder staan.

5. Ik begreep het vrij snel, eerst examine en dan talk to.

6. Het ziet er goed uit, je bent goed op weg. Als tip alleen nog kijken naar hoe je de tekst schrijft zodat het goed bij de game past.
