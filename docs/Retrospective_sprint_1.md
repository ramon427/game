## Teamontwikkeling
Forming: In deze fase waren we bezig met kennismaken en het vaststellen van onze plannen.

Storming: Naarmate we meer met elkaar werkten, ging het steeds soepeler. Doordat we elkaar leerde kennen. Ideeen werden makkelijk uitgewisseld.

Norming: Doordat we nu een stuk verder waren in het project werden er nu veel meer vragen gesteld en dus ook meer hulp verleend.

Performing: Wij bevinden ons nu in presetatie, omdat we de code base nu begrijpen, de kamers geconnect zijn, we elkaar goed kennen waardoor we gemotiveerd aan de slag kunnen gaan.
